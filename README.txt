Benjamin Peterson and Cassie Sterns Final Project

Project Name: Minigit

Description: This project is a utility for tracking and backing up changes in source code.

Primary Features:
- Add file: add a file to the current working commit
- Remove file: remove file from the current working commit
- Commit: save the status of all of the current files added and create a new working commit
- Checkout: restore data from a previous commit

Additional Features:
- Status and Difference in file detection (prints line where the two files differ)
